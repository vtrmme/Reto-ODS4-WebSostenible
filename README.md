# Proyecto Web: EduConnect - ODS 4

Este proyecto es una propuesta tecnológica para mejorar la calidad educativa desde el Desarrollo de Aplicaciones Web.

## 1. Análisis del Problema y Sostenibilidad (Alumno A)

### El "Bug" Educativo

> El aprendizaje colaborativo entre alumnos carece de un espacio digital propio dentro del
> entorno escolar. Las plataformas existentes como Moodle están orientadas al profesor,
> dejando sin herramientas a los estudiantes para organizarse, compartir apuntes o resolver
> dudas entre ellos. Esto genera aislamiento académico, especialmente en alumnos con menos
> recursos que no pueden permitirse academias o clases particulares. La brecha digital se
> amplía cuando estas plataformas son pesadas, poco accesibles y requieren conexiones
> rápidas que no todos tienen.

### Nuestro "Parche" Sostenible

- [ ] Medida 1: Imágenes comprimidas y carga diferida (lazy loading) para funcionar bien
      en conexiones lentas o datos móviles limitados.
- [ ] Medida 2: Diseño accesible con soporte para lectores de pantalla (ARIA labels)
      para alumnos con discapacidad visual.
- [ ] Medida 3: Modo oscuro activable para reducir el consumo de batería en pantallas OLED
      y mejorar la experiencia en condiciones de poca luz.
- [ ] Medida 4: Diseño responsive y ligero, sin dependencias innecesarias, para que
      funcione en dispositivos móviles antiguos o de gama baja.
- [ ] Medida 5: Sin publicidad ni rastreadores de terceros, reduciendo el peso de la
      página y protegiendo la privacidad de los alumnos menores de edad.
